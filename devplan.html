Relational Schema

USERS

user(user_id, username, password_hash, first_name, last_name, created_at, height_feet, height_inches, weight, goal, residence, initial_weight, goal_cals, goal_protein, goal_carbs, goal_fat, goal_burned, gender)

Dependencies: {user_id: everything}


WORKOUTS

workout(workout_id, workout_name, created_by_user)

Dependencies: {workout_id: created_by_user}

tracked_workout(tracked_workout_id, created_by_user, calories_burned, workout_type, workout_name, workout_date)

Dependencies: {tracked_workout_id: created_by_user, calories_burned, workout_type, workout_date)

exercise(exercise_name, equipment_name, target_muscle, demo_link)

Dependencies: {exercise_name: everything}

workout_comprised_of(workout_id, exercise_name, workout_order, sets, reps)

Dependencies: {workout_id, exercise_name: workout_order, sets, reps}

exercise_location(exercise_location_name, location_type)

Dependencies: {exercise_location_name: location_type}

exercise_equipment(equipment_name)

Dependencies: none

equipment_available_at(exercise_location_name, equipment_name)

Dependencies: none


MEALS

meal(meal_id, user_id, meal_name, meal_date, meal_type, meal_time)

Dependencies: {meal_id: everything}

meal_item(food_id, meal_id, servings)

Dependencies: {food_id, meal_id: servings}

food_item(food_id, food_name, calories, protein, carbs, fat, sugar, food_group, sourced_from, meal_time, is_offered, serving_size)

Dependencies: {food_id: everything}

Database and Software Choices

db.cse.nd.edu Server: Setting up a new server is redundant. Also, this is free!

PHP: We are comfortable with this language from this class and past projects. No need to learn a new language.

Python: Easy to run as a cronjob in order to scrape data daily.

Data Sources

Menu and Dining Data: Campus Dining Menus (dining.nd.edu)

Exercise Equipment Data: RecSports Contact

User Data: Users create their own accounts and input their own data. We will create sample users to test the software and functions.

Exercise Data: Exercise data was obtained from our own brainstorming as well as these sources:

 healthline.com

 self.com

 medicalnewstoday.com
