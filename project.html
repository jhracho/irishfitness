Irish Fitness Project Documentation | CSE 30246

Project Description

Irish Fitness is a web application that will eventually (hopefully) transform into an iOS and Android app as well.

The goal of Irish Fitness is to allow users to build healthy and nutritious meals using Campus Dining menus. These menus, including North and South Dining Halls and flex point options, will be parsed from dining.nd.edu.

Users will also be able to build a workout plan based on exercise equipment available to them in places such as Duncan, The Rock, or their dorm gym. There will be a list of example exercises that the user can choose from, as well as links to instructional videos to show how each exercise is performed. We will seek out a partnership with RecSports to provide any additional functionality that they would like and obtain any advice that they have.

It is also possible that we seek out help from the IDEA Center on campus to maximize the potential of Irish Fitness.

Project Usefulness

Our target audience includes Notre Dame students and faculty interested in leading a healthy lifestyle while on campus. Similar fitness websites exist (eg. MyFitnessPal), but these apps do not work well with Campus Dining menus and limited availability of workout equipment on campus. Irish Fitness seeks to bridge the gap between Notre Dame and fitness tracking.

Data Realness

Our data includes meals offered throughout campus as well as exercise equipment at the various gyms. We will attempt to obtain this data through communication with Campus Dining and RecSports.

We will also have a database of users, where they can track their goals, including target weight, target macros, and exercise preference.

Users will also have the opportunity to track their meals to better understand how what they eat is affeting their health.

Functions

Basic Functions

Insert

Users can make accounts within Irish Fitness. Users will then be able to insert meal and workout data that they wish to save.

Update

Users can update their information, such as weight and goals as well as update meal and workout data previously entered.

Delete

Admins will have the power to delete users. Users will be able to delete any previous data entered.

Search

Users can search through their meal or workout data to reach what they would like to see.

Advanced Functions

Menu Parsing: Each day, Campus Dining's menus will be parsed for meal items and, if the item is not already in the database, it will be added to the database. Then, each item being offered during the current day will be selected from the database and shown to the user.

Running Route Overlay: Users will have the option to go on an outdoor run on a scripted path. The user will insert a starting point and a desired distance; Irish Fitness will then output a running route throughout campus overlayed on the campus map using the desired distance and starting point.

Custom Recipe/Meal Input: Users will be able to create and save recipes to add to their meal plan for the day. They will also be able to add meals by number of calories or macros. This will allow for easier use when a user wishes to eat somewhere off campus.
